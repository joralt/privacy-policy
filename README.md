Política de Privacidad – CineptiaIA

Última actualización: 15 de diciembre de 2025

Esta Política de Privacidad explica cómo CineptiaIA (“la app”, “nosotros”) recopila, usa y protege tu información cuando utilizas nuestra aplicación móvil.

Si no estás de acuerdo con esta política, por favor no uses la app.
1. Alcance

Esta política aplica a:

La aplicación móvil CineptiaIA (Android).

Servicios relacionados (por ejemplo, autenticación y base de datos).

2. Información que recopilamos
2.1 Información que tú proporcionas

Cuenta e inicio de sesión: para usar CineptiaIA es necesario iniciar sesión. Normalmente esto incluye correo electrónico y datos básicos asociados a tu cuenta (según el proveedor de autenticación).

Preferencias y actividad dentro de la app: por ejemplo:

películas marcadas como favoritas

interacciones como “me gusta/no” para entrenar recomendaciones

listas o selecciones dentro de la app (si las usas)

Estado de ánimo (Mood): si decides registrarlo, guardamos tu mood, y variables relacionadas (por ejemplo etiqueta, energía, valencia, géneros sugeridos/evitar), para personalizar recomendaciones.

2.2 Información recopilada automáticamente

Identificadores técnicos y de seguridad: datos necesarios para operar el servicio (p. ej. identificadores de autenticación de Firebase, timestamps, logs de errores).

Datos de uso básicos: acciones necesarias para que la app funcione (por ejemplo, cuándo se crea/actualiza un registro).
No buscamos recopilar información innecesaria.

2.3 Contenido generado con IA (si lo usas)

Cuando usas funciones de recomendaciones o chat:

Procesamos el texto que escribes (por ejemplo “recomiéndame una comedia corta”) para generar una respuesta.

La app puede enviar ese texto a un servicio de IA a través de nuestro backend (Cloud Functions) para evitar exponer claves en el dispositivo.

No debes enviar datos sensibles (documentos, contraseñas, información médica, financiera, etc.) por el chat.

3. Cómo usamos tu información

Usamos tu información para:

Crear y administrar tu cuenta y sesión.

Guardar tus favoritos y preferencias.

Generar recomendaciones personalizadas (incluyendo el uso de tu mood si lo registras).

Mejorar la experiencia de la app (rendimiento, corrección de errores).

Proteger la seguridad del servicio y prevenir abuso.

4. Servicios de terceros

CineptiaIA puede usar servicios de terceros para operar:

4.1 Firebase (Google)

Usamos Firebase para:

Autenticación

Base de datos (Firestore)

Almacenamiento (Storage) (si en algún punto subes un avatar u otro recurso)

Cloud Functions (para lógica backend como proxy/IA)

Firebase puede procesar datos necesarios para prestar el servicio. El tratamiento se rige por las políticas de Google/Firebase.

4.2 TMDb (The Movie Database)

Usamos la API de TMDb para mostrar información de películas (títulos, posters, ratings, etc.).
TMDb puede recibir información técnica de la solicitud (por ejemplo, parámetros de búsqueda) cuando la app consulta su API.

Aviso: CineptiaIA utiliza TMDb pero no está respaldada ni certificada por TMDb.

4.3 Proveedores de IA (si aplica)

Cuando usas chat/recomendaciones con IA, la app puede enviar tu texto a un proveedor de IA mediante nuestro backend.
El objetivo es responder a tu solicitud y mejorar la utilidad del servicio.

5. Base legal / Consentimiento

Al usar CineptiaIA:

Nos autorizas a procesar la información necesaria para brindarte el servicio.

En funciones opcionales (como registrar mood), tú decides si lo ingresas o no.

6. Cómo compartimos tu información

No vendemos tu información personal.

Podemos compartir información solo en estos casos:

Con proveedores de servicio (Firebase, TMDb y/o proveedor de IA) para operar la app.

Por motivos legales si una autoridad competente lo requiere.

Para seguridad: prevenir fraude, abuso, o ataques a nuestros sistemas.

7. Retención de datos

Conservamos tus datos mientras:

Tu cuenta esté activa, o

Sea necesario para operar la app y cumplir obligaciones legales.

Puedes solicitar eliminación (ver sección 10).

8. Seguridad

Aplicamos medidas razonables para proteger tu información:

Reglas de acceso en base de datos (por ejemplo, acceso por usuario autenticado).

Uso de servicios administrados (Firebase).

Separación de secretos en backend (por ejemplo, claves de IA no se guardan en la app).

Aun así, ningún sistema es 100% seguro. Recomendamos:

No compartir tu contraseña.

No enviar datos sensibles por el chat.

9. Menores de edad

CineptiaIA no está dirigida a menores de 13 años (o la edad mínima equivalente en tu país).
Si crees que un menor nos proporcionó datos, contáctanos para eliminarlos.

10. Tus derechos y controles

Dependiendo de tu país, puedes tener derechos como:

Acceder a tus datos

Corregirlos

Solicitar eliminación

Oponerte a ciertos tratamientos

Eliminar datos / cuenta:
Puedes solicitar eliminación de tu cuenta y datos asociados. (El canal de soporte se publicará dentro de la app o en el correo de contacto).

11. Permisos del dispositivo

La app puede solicitar permisos solo si son necesarios para funciones específicas (por ejemplo, galería si subes una imagen).
Si no usas esa función, puedes denegar el permiso.

12. Transferencias internacionales

Tus datos pueden procesarse en servidores ubicados en otros países (por ejemplo, infraestructura cloud). Tomamos medidas razonables para que el tratamiento siga esta política.

13. Cambios a esta política

Podemos actualizar esta política ocasionalmente.
Publicaremos la versión actualizada en esta misma página y actualizaremos la fecha de “Última actualización”.
